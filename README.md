Project-SSA
===========

**Project-SSA** 是一个基于 **RoR** 与 **Node.js** 的类MOOC系统。

###**项目尚未完成，请谨慎使用代码**###

##组成部分##

###**Scuola**###
* 内容管理
* 账号管理

###Speculum###
* HTML5课程播放器
* Node.js后端与播放器保持心跳

###Aether###
* 是否实现有待商榷，通过流媒体的形式进行课程直播，在speculum的帮助下可以进行简单的提问互动。
* **十分占用资源**
